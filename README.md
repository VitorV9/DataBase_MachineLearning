## 📊 DataBase_MachineLearning — Detecção de Fake News com Naive Bayes
Script desenvolvido em Ruby para a extração, limpeza e estruturação de uma base de dados textuais focada em desinformação sobre Inteligência Artificial, gerando o arquivo de entrada nativo para mineração de dados no Weka.

## 📌 Escopo do Trabalho: Trabalho Prático 1 (T1) — Nota 2 (N2).
Tema: Desinformação com/sobre Inteligência Artificial

Quantidade de Dados coletados: A base de dados é composta por 60 notícias balanceadas, sendo 40 classificadas como Fato e 40 como Fake.

Período de Coleta: O período de coleta das publicações foi de fevereiro de 2025 até junho de 2026.

## 🛠️ Tecnologias e Ferramentas
Construção do Dataset: Ruby (v3.x+)
Gerenciador de Dependências: Bundler e json

Algoritmo de Machine Learning: Weka (Interface Gráfica — NaiveBayesMultinomial)
